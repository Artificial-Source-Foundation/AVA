## Usage

Add design mockups here for reference when building UI components.

Supported formats:
- Images (PNG, JPG, SVG)
- Figma exports
- Sketches

## Naming Convention

```
<component>-<variant>.<ext>

Examples:
- chat-input-default.png
- sidebar-sessions.png
- settings-modal.png
```
