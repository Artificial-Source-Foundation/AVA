# Troubleshooting

Use this folder for platform-specific setup and runtime issues that do not belong in the main docs index.

## Guides

| File | Purpose |
|------|---------|
| `tauri-toolchain-checklist.md` | Linux desktop build prerequisites and smoke checks |
| `webkitgtk-rendering.md` | Linux WebKitGTK rendering bugs and workarounds |

## When To Add A File Here

Add a troubleshooting note when all of the following are true:

- the issue is environment-specific or platform-specific
- it is likely to recur for contributors or users
- the fix is operational guidance rather than architecture documentation

Keep these files practical: symptoms, cause, fix, and any relevant file paths.
